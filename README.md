# N-Queen

## Overview
 - To get insight about algorithms in solve problem
 
## Features
 - Uniform-cost search
 - A* with MIN-CONFLICT heuristic
 - Random Restart Hill-climbing
 - Genetic Algorithm
 
## How to use
- Execute script **main.py**
- Input number queens
- Select algorithm to solve.
